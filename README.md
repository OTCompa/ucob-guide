<h1 align="center">UCoB Guide.</h4>

## Getting Started

Get started by running the following commands:

```shell
git clone git@github.com:Yhumi/ucob-guide.git
cd ucob-guide
yarn install
yarn dev
```

Visit `http://localhost:4321/` to view the app.
